## 玩法说明

这是一份精简后的游戏说明：

- **核心目标**：在限定步数内规划最长连线路径，让篮子吞噬材料，利用道具打出高连击获得高分。
- **基础操作**：每次连线必须从篮子所在格子开始，可沿8个方向连接相邻的同色材料；沿原路滑回可撤销，松开手指不代表结束，需点击“出发”按钮才执行路径。
- **动态位移与补全**：点击出发后篮子沿规划路径逐格跳动，吞噬材料并留下空位，空位会随机长出新材料，篮子停在终点作为下一回合起点。
- **特殊道具**  
  - 果冻：连线经过后可重置目标颜色，允许从果冻连向任意其他颜色。  
  - 宝箱：无颜色限制的中转站，连入后可继续连接原有颜色，并提供分数加成。  
  - 墙壁：黑色实心，无法连接或穿过，需绕行。
- **难度等级**  
  - 简单：无限步数，障碍极少，系统会生成保底长路径。  
  - 中等：障碍增加，高频产出宝箱。  
  - 困难：障碍密集，需利用果冻完成极限一波流。
